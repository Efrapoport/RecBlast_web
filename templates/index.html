Reciprocal Blast for the masses

Cloud-Based Large Scale Orthology Detection

RecBlast is a cloud-based web server, which performs large-scale orthology detection across multiple genes and multiple taxa (MGMT) by implementing fast reciprocal protein Blast.
RecBlast can be used via a web server, and is also available as a powerful standalone software which can run on multiple cores simultaneously.
